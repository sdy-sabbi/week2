# ATM Simulation Program using Python

## Project Overview
This project is a console-based ATM Simulation Program developed using Python.  
It simulates basic ATM operations with a **menu-driven interface**, formatted output.

The program is designed to understand Python fundamentals such as:
- Conditional statements
- Loops
- Input handling
- Basic validation
- Formatted console output

---

## Features
- ATM PIN verification
- Balance enquiry
- Deposit money
- Withdraw money
- Change ATM PIN
- Change registered phone number
- Neatly formatted console output
- time delay using `time.sleep(3)`

---

## How to Run the Program
1. Make sure **Python 3** is installed on your system.
2. Save the program file as `atm.py`.
3. Open **Terminal / Command Prompt**.
4. Navigate to the folder containing `atm.py`.
5. Run the program using:

```bash
python atm.py
```
